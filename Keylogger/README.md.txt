# Basic Keylogger

## Description
This project is a basic keylogger built using Python. 
It records keystrokes and saves them into a log file 
for educational cybersecurity purposes.

## Tools & Technologies Used
- Python
- pynput 

## Features
- Captures keystrokes
- Saves logs into file
- Runs in background

## Risk Analysis and Security Implications

Keylogging attacks pose serious cybersecurity risks. Some potential risks include:

- Theft of sensitive information such as passwords and banking details.
- Unauthorized access to personal or organizational accounts.
- Identity theft and financial fraud.
- Corporate data breaches.
- Violation of privacy laws and ethical boundaries.

This project was developed strictly for educational purposes to understand 
how such attacks work and how to defend against them. Understanding keyloggers 
helps cybersecurity professionals design better detection and prevention systems.
